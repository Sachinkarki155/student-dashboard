# 📊 Student Data Analysis Dashboard

## 🚀 Project Overview

This is a simple and interactive **Student Dashboard** built using HTML, CSS, and JavaScript.

It analyzes student data and displays useful insights like:

* 📈 Average Marks
* 🏆 Top 3 Students
* ✅ Pass/Fail Status
* 🎯 Grade System (A, B, C)
* 🔍 Search Functionality
* 📊 Graph Visualization (Chart.js)
